# Docs

Internal documentation for the repo's authoring workflow.

## Layout

- [handoff-briefs/](handoff-briefs/) — templates and worked examples
  for the brief-before-draft pattern used to author skills and
  subagents. See
  [Handoff discipline](../README.md#handoff-discipline) in the
  top-level README for context.

A second directory, `project-instructions/`, holds the author's
personal Claude Desktop project instructions (easier to edit here and
paste into Desktop). It is gitignored and not part of the public repo.
